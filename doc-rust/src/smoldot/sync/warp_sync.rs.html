18//! Warp syncing.
19//!
20//! Warp syncing makes it possible to very quickly reach the currently finalized block of a chain.
21//! No attempt is made at verifying blocks. The algorithm assumes that the currently finalized
22//! block is valid, as the chain is likely bricked if this is not the case.
23//!
24//! # Long range attack vulnerability
25//!
26//! Warp syncing is particularly vulnerable to what is called long range attacks.
27//!
28//! The authorities allowed to finalize blocks can generate multiple proofs of finality for
29//! multiple different blocks of the same height. In other words, they can finalize more than one
30//! chain at a time.
31//!
32//! Finalizing multiple chains at the same time is called an equivocation. Client implementations
33//! detect equivocations and report them to the runtime. While this is out of scope of the client,
34//! the runtime then typically provides a mechanism that punishes the validators that have
35//! equivocated.
36//! However, this defense mechanism is flawed in case when a long time passes between the
37//! multiple finality proofs generated by the same validators. Clients cannot hold an unlimited
38//! amount of information in memory, so they might not detect the equivocation, and even if it is
39//! detected, the punishment might not be enforceable because validators have moved all their
40//! funds.
41//!
42//! In other words, it is possible for two thirds of the validators that were active at a certain
43//! past block N to collude and decide to finalize a different block N, even when N has been
44//! finalized for the first time several weeks or months in the past. When a client then warp
45//! syncs, it can be tricked to consider this alternative block N as the finalized one.
46//!
47//! There is no fool-proof defense against this attack. However, consider the extremely high
48//! investment and high risk for the malicious validators, and the difficulty of pulling off this
49//! attack, it is extremely unlikely to happen in reality.
50//! The aforementioned punishment system is the only defense against this attack, and in order to
51//! be effective, the starting point of the warp syncing shouldn't be too far in the past. How
52//! far exactly depends on the logic of the runtime of the chain.
53//!
54//! # Overview
55//!
56//! The warp syncing algorithm works only if the chain uses Grandpa for its finality.
57//! It consists in the following steps:
58//!
59//! - Downloading a warp sync proof from a source. This proof contains a list of *fragments*. Each
60//! fragment represents a change in the list of Grandpa authorities, and a list of signatures of
61//! the previous authorities that certify that this change is correct.
62//! - Verifying the fragments. Each fragment that is successfully verified progresses towards
63//! the head of the chain. Even if one fragment is invalid, all the previously-verified
64//! fragments can still be kept, and the warp syncing can resume from there.
65//! - Downloading from a source the runtime code of the final block of the proof.
66//! - Performing some runtime calls in order to obtain the current consensus-related parameters
67//! of the chain. This might require obtaining some storage items, in which case they must also
68//! be downloaded from a source.
69//!
70//! At the end of the syncing, a [`ValidChainInformation`] corresponding to the head of the chain
71//! is yielded.
72//!
73//! # Usage
74//!
75//! Use the [`start_warp_sync()`] function to start a Grandpa warp syncing state machine.
76//!
77//! At any given moment, this state machine holds a list of *sources* that it might use to
78//! download the warp sync proof or the runtime code. Sources must be added and removed by the API
79//! user by calling one of the various `add_source` and `remove_source` functions.
80//!
81//! Sources are identified through a [`SourceId`]. Each source has an opaque so-called "user data"
82//! of type `TSrc` associated to it. The content of this "user data" is at the discretion of the
83//! API user.
84//!
85//! Similarly, at any given moment, this state machine holds a list of requests that concern these
86//! sources. Use [`WarpSync::desired_requests`] to determine which requests will be useful to the
87//! progress of the warp syncing, then use [`WarpSync::add_request`] to update the state machine
88//! with a newly-started request.
89//!
90//! Use [`WarpSync::process_one`] in order to run verifications of the payloads that have
91//! previously been downloaded.
92//!

120/// The configuration for [`start_warp_sync()`].
121#[derive(Debug)]
122pub struct Config {
123    /// The chain information of the starting point of the warp syncing.
124    pub start_chain_information: ValidChainInformation,
125
126    /// Number of bytes used when encoding/decoding the block number. Influences how various data
127    /// structures should be parsed.
128    pub block_number_bytes: usize,
129
130    /// The initial capacity of the list of sources.
131    pub sources_capacity: usize,
132
133    /// The initial capacity of the list of requests.
134    pub requests_capacity: usize,
135
136    /// Known valid Merkle value and storage value combination for the `:code` key.
137    ///
138    /// If provided, the warp syncing algorithm will first fetch the Merkle value of `:code`, and
139    /// if it matches the Merkle value provided in the hint, use the storage value in the hint
140    /// instead of downloading it. If the hint doesn't match, an extra round-trip will be needed,
141    /// but if the hint matches it saves a big download.
142    pub code_trie_node_hint: Option<ConfigCodeTrieNodeHint>,
143
144    /// Number of warp sync fragments after which the state machine will pause downloading new
145    /// ones until the ones that have been downloaded are verified.
146    ///
147    /// A too low value will cause stalls, while a high value will use more memory and runs the
148    /// risk of wasting more bandwidth in case the downloaded fragments need to be thrown away.
149    pub num_download_ahead_fragments: usize,
150
151    /// If the height of the current local finalized block is `N`, the warp sync state machine
152    /// will not attempt to warp sync to blocks whose height inferior or equal to `N + k` where
153    /// `k` is the value in this field.
154    ///
155    /// Because warp syncing is a relatively expensive process, it is not worth performing it
156    /// between two blocks that are too close to each other.
157    ///
158    /// The ideal value of this field depends on the block production rate and the time it takes
159    /// to answer requests.
160    pub warp_sync_minimum_gap: usize,
161
162    /// If `true`, the body of the warp sync target will be downloaded before the warp sync
163    /// finishes.
164    /// Determines whether [`RuntimeInformation::finalized_body`] is `Some`.
165    pub download_block_body: bool,
166
167    /// If `true`, all the storage proofs and call proofs necessary in order to compute the chain
168    /// information of the warp synced block will be downloaded. If `false`, the finality
169    /// information of the warp synced block is inferred from the warp sync fragments instead.
170    pub download_all_chain_information_storage_proofs: bool,
171}

186/// Initializes the warp sync state machine.
187///
188/// On error, returns the [`ValidChainInformation`] that was provided in the configuration.
189pub fn start_warp_sync<TSrc, TRq>(
190    config: Config,
191) -> Result<WarpSync<TSrc, TRq>, (ValidChainInformation, WarpSyncInitError)> {
192    match config.start_chain_information.as_ref().finality {
193        // TODO: we make sure that `finalized_scheduled_change` is `None` because it seems complicated to support, but ideally it would be supported
194        ChainInformationFinalityRef::Grandpa {
195            finalized_scheduled_change: None,
196            ..
197        } => {}
198        _ => {
199            return Err((
200                config.start_chain_information,
201                WarpSyncInitError::NotGrandpa,
202            ));
203        }
204    }
205
206    match config.start_chain_information.as_ref().consensus {
207        ChainInformationConsensusRef::Babe { .. } | ChainInformationConsensusRef::Aura { .. } => {}
208        ChainInformationConsensusRef::Unknown => {
209            return Err((
210                config.start_chain_information,
211                WarpSyncInitError::UnknownConsensus,
212            ));
213        }
214    }
215
216    let warped_header = config
217        .start_chain_information
218        .as_ref()
219        .finalized_block_header
220        .scale_encoding_vec(config.block_number_bytes);
221
222    Ok(WarpSync {
223        warped_header_number: config
224            .start_chain_information
225            .as_ref()
226            .finalized_block_header
227            .number,
228        warped_header_state_root: *config
229            .start_chain_information
230            .as_ref()
231            .finalized_block_header
232            .state_root,
233        warped_header_extrinsics_root: *config
234            .start_chain_information
235            .as_ref()
236            .finalized_block_header
237            .extrinsics_root,
238        warped_header_hash: header::hash_from_scale_encoded_header(&warped_header),
239        warped_header,
240        warped_finality: config.start_chain_information.as_ref().finality.into(),
241        warped_block_ty: WarpedBlockTy::AlreadyVerified,
242        runtime_calls: runtime_calls_default_value(
243            config.start_chain_information.as_ref().consensus,
244        ),
245        verified_chain_information: config.start_chain_information,
246        code_trie_node_hint: config.code_trie_node_hint,
247        num_download_ahead_fragments: config.num_download_ahead_fragments,
248        warp_sync_minimum_gap: config.warp_sync_minimum_gap,
249        block_number_bytes: config.block_number_bytes,
250        download_all_chain_information_storage_proofs: config
251            .download_all_chain_information_storage_proofs,
252        sources: slab::Slab::with_capacity(config.sources_capacity),
253        sources_by_finalized_height: BTreeSet::new(),
254        in_progress_requests: slab::Slab::with_capacity(config.requests_capacity),
255        in_progress_requests_by_source: BTreeSet::new(),
256        warp_sync_fragments_download: None,
257        verify_queue: VecDeque::new(),
258        runtime_download: RuntimeDownload::NotStarted {
259            hint_doesnt_match: false,
260        },
261        body_download: if config.download_block_body {
262            BodyDownload::NotStarted
263        } else {
264            BodyDownload::NotNeeded
265        },
266    })
267}

538impl<TSrc, TRq> WarpSync<TSrc, TRq> {
539    /// Returns the value that was initially passed in [`Config::block_number_bytes`].
540    pub fn block_number_bytes(&self) -> usize {
541        self.block_number_bytes
542    }
543
544    /// Returns the chain information that is considered verified.
545    pub fn as_chain_information(&'_ self) -> ValidChainInformationRef<'_> {
546        // Note: after verifying a warp sync fragment, we are certain that the header targeted by
547        // this fragment is indeed part of the chain. However, this is not enough in order to
548        // produce a full chain information struct. Such struct can only be produced after the
549        // entire warp syncing has succeeded. If if it still in progress, all we can return is
550        // the starting point.
551        (&self.verified_chain_information).into()
552    }
553
554    /// Modifies the chain information known to be valid.
555    pub fn set_chain_information(&mut self, chain_information: ValidChainInformationRef) {
556        // The implementation simply resets the state machine, apart from the fragment download
557        // and verification queue.
558        // TODO: what if the new chain doesn't support grandpa?
559        if self.warped_header_number <= chain_information.as_ref().finalized_block_header.number {
560            self.warped_header = chain_information
561                .as_ref()
562                .finalized_block_header
563                .scale_encoding_vec(self.block_number_bytes);
564            self.warped_header_hash = chain_information
565                .as_ref()
566                .finalized_block_header
567                .hash(self.block_number_bytes);
568            self.warped_header_state_root =
569                *chain_information.as_ref().finalized_block_header.state_root;
570            self.warped_header_extrinsics_root = *chain_information
571                .as_ref()
572                .finalized_block_header
573                .extrinsics_root;
574            self.warped_header_number = chain_information.as_ref().finalized_block_header.number;
575            self.warped_finality = chain_information.as_ref().finality.into();
576            self.warped_block_ty = WarpedBlockTy::AlreadyVerified;
577
578            self.verified_chain_information = chain_information.into();
579            self.runtime_calls =
580                runtime_calls_default_value(self.verified_chain_information.as_ref().consensus);
581
582            self.runtime_download = RuntimeDownload::NotStarted {
583                hint_doesnt_match: false,
584            };
585
586            if !matches!(self.body_download, BodyDownload::NotNeeded) {
587                self.body_download = BodyDownload::NotStarted;
588            }
589        }

592    /// Returns the current status of the warp syncing.
593    pub fn status(&'_ self) -> Status<'_, TSrc> {
594        match &self.runtime_download {
595            RuntimeDownload::NotStarted { .. } => {
596                let finalized_block_hash = self.warped_header_hash;
597
598                let source_id =
599                    if let Some(warp_sync_fragments_download) = self.warp_sync_fragments_download {
600                        Some(
601                            self.in_progress_requests
602                                .get(warp_sync_fragments_download.0)
603                                .unwrap()
604                                .0,
605                        )
606                    } else {
607                        self.verify_queue.back().and_then(|f| f.downloaded_source)
608                    };
609
610                Status::Fragments {
611                    source: source_id.map(|id| (id, &self.sources[id.0].user_data)),
612                    finalized_block_hash,
613                    finalized_block_number: self.warped_header_number,
614                }
615            }
616            _ => Status::ChainInformation {
617                finalized_block_hash: self.warped_header_hash,
618                finalized_block_number: self.warped_header_number,
619            },
620        }
621    }

658    /// Removes a source from the list of sources. In addition to the user data associated to this
659    /// source, also returns a list of requests that were in progress concerning this source. These
660    /// requests are now considered obsolete.
661    ///
662    /// # Panic
663    ///
664    /// Panics if the [`SourceId`] is invalid.
665    ///
666    pub fn remove_source(
667        &mut self,
668        to_remove: SourceId,
669    ) -> (TSrc, impl Iterator<Item = (RequestId, TRq)>) {
670        debug_assert!(self.sources.contains(to_remove.0));
671        let removed = self.sources.remove(to_remove.0);
672        let _was_in = self
673            .sources_by_finalized_height
674            .remove(&(removed.finalized_block_height, to_remove));
675        debug_assert!(_was_in);
676        debug_assert!(self.sources.len() >= self.sources_by_finalized_height.len());
677
678        // We make sure to not leave invalid source IDs in the state of `self`.
679        // TODO: O(n)
680        for item in &mut self.verify_queue {
681            if item.downloaded_source == Some(to_remove) {
682                item.downloaded_source = None;
683            }
684        }
685        if let RuntimeDownload::NotVerified {
686            downloaded_source, ..
687        } = &mut self.runtime_download
688        {
689            if *downloaded_source == Some(to_remove) {
690                *downloaded_source = None;
691            }
692        }
693        if let BodyDownload::Downloaded {
694            downloaded_source, ..
695        } = &mut self.body_download
696        {
697            if *downloaded_source == Some(to_remove) {
698                *downloaded_source = None;
699            }
700        }
701        for (_, call_proof) in &mut self.runtime_calls {
702            if let CallProof::Downloaded {
703                downloaded_source, ..
704            } = call_proof
705            {
706                if *downloaded_source == Some(to_remove) {
707                    *downloaded_source = None;
708                }
709            }
710        }
711
712        let obsolete_requests_indices = self
713            .in_progress_requests_by_source
714            .range((to_remove, RequestId(usize::MIN))..=(to_remove, RequestId(usize::MAX)))
715            .map(|(_, rq_id)| rq_id.0)
716            .collect::<Vec<_>>();
717        let mut obsolete_requests = Vec::with_capacity(obsolete_requests_indices.len());
718        for index in obsolete_requests_indices {
719            let (_, user_data, _) = self.in_progress_requests.remove(index);
720            self.in_progress_requests_by_source
721                .remove(&(to_remove, RequestId(index)));
722            if self.warp_sync_fragments_download == Some(RequestId(index)) {
723                self.warp_sync_fragments_download = None;
724            }
725            for call in self.runtime_calls.values_mut() {
726                if matches!(call, CallProof::Downloading(rq_id) if *rq_id == RequestId(index)) {
727                    *call = CallProof::NotStarted;
728                }
729            }
730            if let RuntimeDownload::Downloading {
731                request_id,
732                hint_doesnt_match,
733            } = &mut self.runtime_download
734            {
735                if *request_id == RequestId(index) {
736                    self.runtime_download = RuntimeDownload::NotStarted {
737                        hint_doesnt_match: *hint_doesnt_match,
738                    };
739                }
740            }
741            if let BodyDownload::Downloading { request_id } = &mut self.body_download {
742                if *request_id == RequestId(index) {
743                    self.body_download = BodyDownload::NotStarted;
744                }
745            }
746            obsolete_requests.push((RequestId(index), user_data));
747        }
748
749        (removed.user_data, obsolete_requests.into_iter())
750    }

752    /// Sets the finalized block height of the given source.
753    ///
754    /// # Panic
755    ///
756    /// Panics if `source_id` is invalid.
757    ///
758    pub fn set_source_finality_state(&mut self, source_id: SourceId, finalized_block_height: u64) {
759        let stored_height = &mut self.sources[source_id.0].finalized_block_height;
760
761        // Small optimization. No need to do anything more if the block doesn't actuall change.
762        if *stored_height == finalized_block_height {
763            return;
764        }
765
766        // Note that if the new finalized block is below the former one (which is not something
767        // that is ever supposed to happen), we should in principle cancel the requests
768        // targeting that source that require a specific block height. In practice, however,
769        // we don't care as again this isn't supposed to ever happen. While ongoing requests
770        // might fail as a result, this is handled the same way as a regular request failure.
771
772        let _was_in = self
773            .sources_by_finalized_height
774            .remove(&(*stored_height, source_id));
775        debug_assert!(_was_in);
776        let _inserted = self
777            .sources_by_finalized_height
778            .insert((finalized_block_height, source_id));
779        debug_assert!(_inserted);
780
781        *stored_height = finalized_block_height;
782    }

796    /// Returns a list of requests that should be started in order to drive the warp syncing
797    /// process to completion.
798    ///
799    /// Once a request that matches a desired request is added through
800    /// [`WarpSync::add_request`], it is no longer returned by this function.
801    pub fn desired_requests(&self) -> impl Iterator<Item = (SourceId, &TSrc, DesiredRequest)> {
802        // If we are in the fragments download phase, return a fragments download request.
803        let mut desired_warp_sync_request = if self.warp_sync_fragments_download.is_none() {
804            if self.verify_queue.iter().fold(0, |sum, entry| {
805                sum + entry.fragments.len() - entry.next_fragment_to_verify_index
806            }) < self.num_download_ahead_fragments
807            {
808                // Block hash to request.
809                let start_block_hash = self
810                    .verify_queue
811                    .back()
812                    .and_then(|entry| entry.fragments.last())
813                    .map(|fragment| {
814                        header::hash_from_scale_encoded_header(&fragment.scale_encoded_header)
815                    })
816                    .unwrap_or(self.warped_header_hash);
817
818                // Calculate the block number at the tail of the verify queue.
819                // Contains `None` if the verify queue has a problem such as an indecodable header.
820                // In that situation, we don't start any new request and wait for the verify
821                // queue to empty itself.
822                let verify_queue_tail_block_number = self
823                    .verify_queue
824                    .back()
825                    .map(|entry| {
826                        entry
827                            .fragments
828                            .last()
829                            .and_then(|fragment| {
830                                header::decode(
831                                    &fragment.scale_encoded_header,
832                                    self.block_number_bytes,
833                                )
834                                .ok()
835                            })
836                            .map(|header| header.number)
837                    })
838                    .unwrap_or(Some(self.warped_header_number));
839                let warp_sync_minimum_gap = self.warp_sync_minimum_gap;
840
841                if let Some(verify_queue_tail_block_number) = verify_queue_tail_block_number {
842                    // Combine the request with every single available source.
843                    either::Left(self.sources.iter().filter_map(move |(src_id, src)| {
844                        if src.finalized_block_height
845                            <= verify_queue_tail_block_number.saturating_add(
846                                u64::try_from(warp_sync_minimum_gap).unwrap_or(u64::MAX),
847                            )
848                        {
849                            return None;
850                        }
851
852                        Some((
853                            SourceId(src_id),
854                            &src.user_data,
855                            DesiredRequest::WarpSyncRequest {
856                                block_hash: start_block_hash,
857                            },
858                        ))
859                    }))
860                } else {
861                    either::Right(iter::empty())
862                }
863            } else {
864                either::Right(iter::empty())
865            }
866        } else {
867            either::Right(iter::empty())
868        }
869        .peekable();
870
871        // If we are in the appropriate phase, and we are not currently downloading the runtime,
872        // return a runtime download request.
873        let desired_runtime_parameters_get = if let (
874            WarpedBlockTy::Normal,
875            RuntimeDownload::NotStarted { hint_doesnt_match },
876            None,
877            true,
878            None,
879        ) = (
880            &self.warped_block_ty,
881            &self.runtime_download,
882            self.warp_sync_fragments_download,
883            self.verify_queue.is_empty(),
884            desired_warp_sync_request.peek(),
885        ) {
886            let code_key_to_request = if let (false, Some(hint)) =
887                (*hint_doesnt_match, self.code_trie_node_hint.as_ref())
888            {
889                Cow::Owned(
890                    trie::nibbles_to_bytes_truncate(
891                        hint.closest_ancestor_excluding.iter().copied(),
892                    )
893                    .collect::<Vec<_>>(),
894                )
895            } else {
896                Cow::Borrowed(&b":code"[..])
897            };
898
899            // Sources are ordered by increasing finalized block height, in order to
900            // have the highest chance for the block to not be pruned.
901            let sources_with_block = self
902                .sources_by_finalized_height
903                .range((self.warped_header_number, SourceId(usize::MIN))..)
904                .map(|(_, src_id)| src_id);
905
906            either::Left(sources_with_block.map(move |source_id| {
907                (
908                    *source_id,
909                    &self.sources[source_id.0].user_data,
910                    DesiredRequest::StorageGetMerkleProof {
911                        block_hash: self.warped_header_hash,
912                        state_trie_root: self.warped_header_state_root,
913                        keys: vec![code_key_to_request.to_vec(), b":heappages".to_vec()],
914                    },
915                )
916            }))
917        } else {
918            either::Right(iter::empty())
919        };
920
921        // If we are in the appropriate phase, and we are not currently downloading the body of
922        // the block, return a runtime download request.
923        let desired_body_download =
924            if let (WarpedBlockTy::Normal, BodyDownload::NotStarted, None, true, None) = (
925                &self.warped_block_ty,
926                &self.body_download,
927                self.warp_sync_fragments_download,
928                self.verify_queue.is_empty(),
929                desired_warp_sync_request.peek(),
930            ) {
931                // Sources are ordered by increasing finalized block height, in order to
932                // have the highest chance for the block to not be pruned.
933                let sources_with_block = self
934                    .sources_by_finalized_height
935                    .range((self.warped_header_number, SourceId(usize::MIN))..)
936                    .map(|(_, src_id)| src_id);
937
938                either::Left(sources_with_block.map(move |source_id| {
939                    (
940                        *source_id,
941                        &self.sources[source_id.0].user_data,
942                        DesiredRequest::BlockBodyDownload {
943                            block_hash: self.warped_header_hash,
944                            block_number: self.warped_header_number,
945                            extrinsics_root: self.warped_header_extrinsics_root,
946                        },
947                    )
948                }))
949            } else {
950                either::Right(iter::empty())
951            };
952
953        // Return the list of runtime calls indicated by the chain information builder state
954        // machine.
955        let desired_call_proofs = if matches!(self.warped_block_ty, WarpedBlockTy::Normal)
956            && self.warp_sync_fragments_download.is_none()
957            && self.verify_queue.is_empty()
958            && desired_warp_sync_request.peek().is_none()
959        {
960            either::Left(
961                self.runtime_calls
962                    .iter()
963                    .filter(|(_, v)| matches!(v, CallProof::NotStarted))
964                    .map(|(call, _)| DesiredRequest::RuntimeCallMerkleProof {
965                        block_hash: self.warped_header_hash,
966                        function_name: call.function_name().into(),
967                        parameter_vectored: Cow::Owned(call.parameter_vectored_vec()),
968                    })
969                    .flat_map(move |request_detail| {
970                        // Sources are ordered by increasing finalized block height, in order to
971                        // have the highest chance for the block to not be pruned.
972                        let sources_with_block = self
973                            .sources_by_finalized_height
974                            .range((self.warped_header_number, SourceId(usize::MIN))..)
975                            .map(|(_, src_id)| src_id);
976
977                        sources_with_block.map(move |source_id| {
978                            (
979                                *source_id,
980                                &self.sources[source_id.0].user_data,
981                                request_detail.clone(),
982                            )
983                        })
984                    }),
985            )
986        } else {
987            either::Right(iter::empty())
988        };
989
990        // Chain all these demanded requests together.
991        desired_warp_sync_request
992            .chain(desired_runtime_parameters_get)
993            .chain(desired_body_download)
994            .chain(desired_call_proofs)
995    }

997    /// Inserts a new request in the data structure.
998    ///
999    /// > **Note**: The request doesn't necessarily have to match a request returned by
1000    /// >           [`WarpSync::desired_requests`].
1001    ///
1002    /// # Panic
1003    ///
1004    /// Panics if the [`SourceId`] is out of range.
1005    ///
1006    pub fn add_request(
1007        &mut self,
1008        source_id: SourceId,
1009        user_data: TRq,
1010        detail: RequestDetail,
1011    ) -> RequestId {
1012        assert!(self.sources.contains(source_id.0));
1013
1014        let request_slot = self.in_progress_requests.vacant_entry();
1015        let request_id = RequestId(request_slot.key());
1016
1017        match (&detail, &mut self.runtime_download, &mut self.body_download) {
1018            (RequestDetail::WarpSyncRequest { block_hash }, _, _)
1019                if self.warp_sync_fragments_download.is_none()
1020                    && *block_hash
1021                        == self
1022                            .verify_queue
1023                            .back()
1024                            .and_then(|entry| entry.fragments.last())
1025                            .map(|fragment| {
1026                                header::hash_from_scale_encoded_header(
1027                                    &fragment.scale_encoded_header,
1028                                )
1029                            })
1030                            .unwrap_or(self.warped_header_hash) =>
1031            {
1032                self.warp_sync_fragments_download = Some(request_id);
1033            }
1034            (
1035                RequestDetail::BlockBodyDownload {
1036                    block_hash,
1037                    block_number,
1038                },
1039                _,
1040                BodyDownload::NotStarted,
1041            ) => {
1042                if self.sources[source_id.0].finalized_block_height >= self.warped_header_number
1043                    && *block_number == self.warped_header_number
1044                    && *block_hash == self.warped_header_hash
1045                {
1046                    self.body_download = BodyDownload::Downloading { request_id };
1047                }
1048            }
1049            (
1050                RequestDetail::StorageGetMerkleProof { block_hash, keys },
1051                RuntimeDownload::NotStarted { hint_doesnt_match },
1052                _,
1053            ) => {
1054                let code_key_to_request = if let (false, Some(hint)) =
1055                    (*hint_doesnt_match, self.code_trie_node_hint.as_ref())
1056                {
1057                    Cow::Owned(
1058                        trie::nibbles_to_bytes_truncate(
1059                            hint.closest_ancestor_excluding.iter().copied(),
1060                        )
1061                        .collect::<Vec<_>>(),
1062                    )
1063                } else {
1064                    Cow::Borrowed(&b":code"[..])
1065                };
1066
1067                if self.sources[source_id.0].finalized_block_height >= self.warped_header_number
1068                    && *block_hash == self.warped_header_hash
1069                    && keys.iter().any(|k| *k == *code_key_to_request)
1070                    && keys.iter().any(|k| k == b":heappages")
1071                {
1072                    self.runtime_download = RuntimeDownload::Downloading {
1073                        hint_doesnt_match: *hint_doesnt_match,
1074                        request_id,
1075                    };
1076                }
1077            }
1078            (
1079                RequestDetail::RuntimeCallMerkleProof {
1080                    block_hash,
1081                    function_name,
1082                    parameter_vectored,
1083                },
1084                _,
1085                _,
1086            ) => {
1087                for (info, status) in &mut self.runtime_calls {
1088                    if matches!(status, CallProof::NotStarted)
1089                        && self.sources[source_id.0].finalized_block_height
1090                            >= self.warped_header_number
1091                        && *block_hash == self.warped_header_hash
1092                        && function_name == info.function_name()
1093                        && parameters_equal(parameter_vectored, info.parameter_vectored())
1094                    {
1095                        *status = CallProof::Downloading(request_id);
1096                        break;
1097                    }
1098                }
1099            }
1100            _ => {}
1101        }
1102
1103        request_slot.insert((source_id, user_data, detail));
1104        let _was_inserted = self
1105            .in_progress_requests_by_source
1106            .insert((source_id, request_id));
1107        debug_assert!(_was_inserted);
1108        request_id
1109    }

1111    /// Removes the given request from the state machine. Returns the user data that was associated
1112    /// to it.
1113    ///
1114    /// > **Note**: The state machine might want to re-start the same request again. It is out of
1115    /// >           the scope of this module to keep track of requests that don't succeed.
1116    ///
1117    /// # Panic
1118    ///
1119    /// Panics if the [`RequestId`] is invalid.
1120    ///
1121    pub fn remove_request(&mut self, id: RequestId) -> TRq {
1122        if self.warp_sync_fragments_download == Some(id) {
1123            self.warp_sync_fragments_download = None;
1124        }
1125
1126        for call in self.runtime_calls.values_mut() {
1127            if matches!(call, CallProof::Downloading(rq_id) if *rq_id == id) {
1128                *call = CallProof::NotStarted;
1129            }
1130        }
1131
1132        if let RuntimeDownload::Downloading {
1133            request_id,
1134            hint_doesnt_match,
1135        } = &mut self.runtime_download
1136        {
1137            if *request_id == id {
1138                self.runtime_download = RuntimeDownload::NotStarted {
1139                    hint_doesnt_match: *hint_doesnt_match,
1140                }
1141            }
1142        }
1143
1144        if let BodyDownload::Downloading { request_id } = &mut self.body_download {
1145            if *request_id == id {
1146                self.body_download = BodyDownload::NotStarted;
1147            }
1148        }
1149
1150        let (source_id, user_data, _) = self.in_progress_requests.remove(id.0);
1151        let _was_removed = self.in_progress_requests_by_source.remove(&(source_id, id));
1152        debug_assert!(_was_removed);
1153        user_data
1154    }

1166    /// Injects a body download and removes the given request from the state machine.
1167    /// Returns the user data that was associated to it.
1168    ///
1169    /// # Panic
1170    ///
1171    /// Panics if the [`RequestId`] is invalid.
1172    /// Panics if the [`RequestId`] doesn't correspond to a body download request.
1173    ///
1174    pub fn body_download_response(&mut self, id: RequestId, body: Vec<Vec<u8>>) -> TRq {
1175        // Remove the request from the list, obtaining its user data.
1176        // If the request corresponds to the runtime parameters we're looking for, the function
1177        // continues below, otherwise we return early.
1178        let (source_id, user_data) =
1179            match (self.in_progress_requests.remove(id.0), &self.body_download) {
1180                ((source_id, user_data, _), BodyDownload::Downloading { request_id })
1181                    if *request_id == id =>
1182                {
1183                    (source_id, user_data)
1184                }
1185                ((source_id, user_data, RequestDetail::BlockBodyDownload { .. }), _) => {
1186                    let _was_removed = self.in_progress_requests_by_source.remove(&(source_id, id));
1187                    debug_assert!(_was_removed);
1188                    return user_data;
1189                }
1190                (
1191                    (
1192                        _,
1193                        _,
1194                        RequestDetail::RuntimeCallMerkleProof { .. }
1195                        | RequestDetail::WarpSyncRequest { .. }
1196                        | RequestDetail::StorageGetMerkleProof { .. },
1197                    ),
1198                    _,
1199                ) => panic!(),
1200            };
1201
1202        self.body_download = BodyDownload::Downloaded {
1203            downloaded_source: Some(source_id),
1204            body,
1205        };
1206
1207        let _was_removed = self.in_progress_requests_by_source.remove(&(source_id, id));
1208        debug_assert!(_was_removed);
1209
1210        user_data
1211    }

1213    /// Injects a Merkle proof and removes the given request from the state machine.
1214    /// Returns the user data that was associated to it.
1215    ///
1216    /// # Panic
1217    ///
1218    /// Panics if the [`RequestId`] is invalid.
1219    /// Panics if the [`RequestId`] doesn't correspond to a storage get request.
1220    ///
1221    pub fn storage_get_response(&mut self, id: RequestId, merkle_proof: Vec<u8>) -> TRq {
1222        // Remove the request from the list, obtaining its user data.
1223        // If the request corresponds to the runtime parameters we're looking for, the function
1224        // continues below, otherwise we return early.
1225        let (source_id, hint_doesnt_match, user_data) = match (
1226            self.in_progress_requests.remove(id.0),
1227            &self.runtime_download,
1228        ) {
1229            (
1230                (source_id, user_data, _),
1231                RuntimeDownload::Downloading {
1232                    request_id,
1233                    hint_doesnt_match,
1234                },
1235            ) if *request_id == id => (source_id, *hint_doesnt_match, user_data),
1236            ((source_id, user_data, RequestDetail::StorageGetMerkleProof { .. }), _) => {
1237                let _was_removed = self.in_progress_requests_by_source.remove(&(source_id, id));
1238                debug_assert!(_was_removed);
1239                return user_data;
1240            }
1241            (
1242                (
1243                    _,
1244                    _,
1245                    RequestDetail::RuntimeCallMerkleProof { .. }
1246                    | RequestDetail::WarpSyncRequest { .. }
1247                    | RequestDetail::BlockBodyDownload { .. },
1248                ),
1249                _,
1250            ) => panic!(),
1251        };
1252
1253        self.runtime_download = RuntimeDownload::NotVerified {
1254            downloaded_source: Some(source_id),
1255            hint_doesnt_match,
1256            trie_proof: merkle_proof,
1257        };
1258
1259        let _was_removed = self.in_progress_requests_by_source.remove(&(source_id, id));
1260        debug_assert!(_was_removed);
1261
1262        user_data
1263    }
1264
1265    /// Injects a response and removes the given request from the state machine. Returns
1266    /// the user data that was associated to it.
1267    ///
1268    /// # Panic
1269    ///
1270    /// Panics if the [`RequestId`] is invalid.
1271    /// Panics if the [`RequestId`] doesn't correspond to a runtime Merkle call proof request.
1272    ///
1273    pub fn runtime_call_merkle_proof_response(
1274        &mut self,
1275        request_id: RequestId,
1276        response: Vec<u8>,
1277    ) -> TRq {
1278        let (source_id, user_data, RequestDetail::RuntimeCallMerkleProof { .. }) =
1279            self.in_progress_requests.remove(request_id.0)
1280        else {
1281            // Wrong request type.
1282            panic!()
1283        };
1284
1285        for call in self.runtime_calls.values_mut() {
1286            if matches!(call, CallProof::Downloading(rq_id) if *rq_id == request_id) {
1287                *call = CallProof::Downloaded {
1288                    downloaded_source: Some(source_id),
1289                    proof: response,
1290                };
1291                break;
1292            }
1293        }
1294
1295        let _was_removed = self
1296            .in_progress_requests_by_source
1297            .remove(&(source_id, request_id));
1298        debug_assert!(_was_removed);
1299
1300        user_data
1301    }

1303    /// Injects a response and removes the given request from the state machine. Returns
1304    /// the user data that was associated to it.
1305    ///
1306    /// If the header of the last fragment of the response is decodable, this function updates
1307    /// the finalized block of the source.
1308    ///
1309    /// # Panic
1310    ///
1311    /// Panics if the [`RequestId`] is invalid.
1312    /// Panics if the [`RequestId`] doesn't correspond to a warp sync request.
1313    ///
1314    // TODO: more zero cost API w.r.t. the fragments
1315    pub fn warp_sync_request_response(
1316        &mut self,
1317        request_id: RequestId,
1318        fragments: Vec<WarpSyncFragment>,
1319        final_set_of_fragments: bool,
1320    ) -> TRq {
1321        let (rq_source_id, user_data) = match self.in_progress_requests.remove(request_id.0) {
1322            (rq_source_id, user_data, RequestDetail::WarpSyncRequest { .. }) => {
1323                (rq_source_id, user_data)
1324            }
1325            (_, _, _) => panic!(),
1326        };
1327
1328        debug_assert!(self.sources.contains(rq_source_id.0));
1329
1330        // Since we send requests only to sources with an appropriate finalized block, we make
1331        // sure that the finalized block of the source that sent the response matches the
1332        // fragments that it sent.
1333        // If we didn't do that, it would be possible for example to warp sync to block 200 while
1334        // believing that the source is only at block 199, and thus the warp syncing would stall.
1335        if let Some(last_header) = fragments
1336            .last()
1337            .and_then(|h| header::decode(&h.scale_encoded_header, self.block_number_bytes).ok())
1338        {
1339            let src_finalized_height = &mut self.sources[rq_source_id.0].finalized_block_height;
1340
1341            let new_height = if final_set_of_fragments {
1342                // If the source indicated that this is the last fragment, then we know that
1343                // it's also equal to their finalized block.
1344                last_header.number
1345            } else {
1346                // If this is not the last fragment, we know that the finalized block of the
1347                // source is *at least* the one provided.
1348                // TODO: could maybe do + gap or something?
1349                cmp::max(*src_finalized_height, last_header.number.saturating_add(1))
1350            };
1351
1352            if *src_finalized_height != new_height {
1353                let _was_in = self
1354                    .sources_by_finalized_height
1355                    .remove(&(*src_finalized_height, rq_source_id));
1356                debug_assert!(_was_in);
1357
1358                *src_finalized_height = new_height;
1359
1360                let _inserted = self
1361                    .sources_by_finalized_height
1362                    .insert((*src_finalized_height, rq_source_id));
1363                debug_assert!(_inserted);
1364            }
1365        }
1366
1367        if self.warp_sync_fragments_download == Some(request_id) {
1368            self.warp_sync_fragments_download = None;
1369
1370            self.verify_queue.push_back(PendingVerify {
1371                final_set_of_fragments,
1372                downloaded_source: Some(rq_source_id),
1373                fragments,
1374                next_fragment_to_verify_index: 0,
1375            });
1376        }
1377
1378        let _was_removed = self
1379            .in_progress_requests_by_source
1380            .remove(&(rq_source_id, request_id));
1381        debug_assert!(_was_removed);
1382
1383        user_data
1384    }
1385
1386    /// Start processing one CPU operation.
1387    ///
1388    /// This function takes ownership of `self` and yields it back after the operation is finished.
1389    // TODO: take a `&mut self` instead of `self` ; requires many changes in all.rs
1390    pub fn process_one(self) -> ProcessOne<TSrc, TRq> {
1391        // If we've downloaded everything that was needed, switch to "build chain information"
1392        // mode.
1393        if matches!(self.runtime_download, RuntimeDownload::Verified { .. })
1394            && matches!(
1395                self.body_download,
1396                BodyDownload::NotNeeded | BodyDownload::Downloaded { .. }
1397            )
1398            && self
1399                .runtime_calls
1400                .values()
1401                .all(|c| matches!(c, CallProof::Downloaded { .. }))
1402        {
1403            return ProcessOne::BuildChainInformation(BuildChainInformation { inner: self });
1404        }
1405
1406        if let RuntimeDownload::NotVerified { .. } = &self.runtime_download {
1407            return ProcessOne::BuildRuntime(BuildRuntime { inner: self });
1408        }
1409
1410        if !self.verify_queue.is_empty() {
1411            return ProcessOne::VerifyWarpSyncFragment(VerifyWarpSyncFragment { inner: self });
1412        }
1413
1414        ProcessOne::Idle(self)
1415    }
1416}

1558impl<TSrc, TRq> VerifyWarpSyncFragment<TSrc, TRq> {
1559    /// Returns the source that has sent the fragments that we are about to verify, and its user
1560    /// data.
1561    ///
1562    /// Returns `None` if the source has been removed since the fragments have been downloaded.
1563    pub fn proof_sender(&self) -> Option<(SourceId, &TSrc)> {
1564        let entry_to_verify = self.inner.verify_queue.front().unwrap();
1565        let source_id = entry_to_verify.downloaded_source?;
1566        Some((source_id, &self.inner.sources[source_id.0].user_data))
1567    }
1568
1569    /// Verify one warp sync fragment.
1570    ///
1571    /// Must be passed a randomly-generated value that is used by the verification process. Note
1572    /// that the verification is still deterministic.
1573    ///
1574    /// On success, returns the block hash and height that have been verified as being part of
1575    /// the chain.
1576    /// On error, returns why the verification has failed. The warp syncing process still
1577    /// continues.
1578    pub fn verify(
1579        mut self,
1580        randomness_seed: [u8; 32],
1581    ) -> (
1582        WarpSync<TSrc, TRq>,
1583        Result<([u8; 32], u64), VerifyFragmentError>,
1584    ) {
1585        // A `VerifyWarpSyncFragment` is only ever created if `verify_queue` is non-empty.
1586        debug_assert!(!self.inner.verify_queue.is_empty());
1587        let fragments_to_verify = self
1588            .inner
1589            .verify_queue
1590            .front_mut()
1591            .unwrap_or_else(|| unreachable!());
1592
1593        // The source has sent an empty list of fragments. This is invalid.
1594        if fragments_to_verify.fragments.is_empty() {
1595            self.inner.verify_queue.pop_front().unwrap();
1596            return (self.inner, Err(VerifyFragmentError::EmptyProof));
1597        }
1598
1599        // Given that the list of fragments is non-empty, we are assuming that there are still
1600        // fragments to verify, otherwise this entry should have been removed in a previous
1601        // iteration.
1602        let fragment_to_verify = fragments_to_verify
1603            .fragments
1604            .get(fragments_to_verify.next_fragment_to_verify_index)
1605            .unwrap_or_else(|| unreachable!());
1606
1607        // It has been checked at the warp sync initialization that the finality algorithm is
1608        // indeed Grandpa.
1609        let chain_information::ChainInformationFinality::Grandpa {
1610            after_finalized_block_authorities_set_id,
1611            finalized_triggered_authorities,
1612            .. // TODO: support finalized_scheduled_change? difficult to implement
1613        } = &mut self.inner.warped_finality
1614        else {
1615            unreachable!()
1616        };
1617
1618        // Decode the header and justification of the fragment.
1619        let fragment_header_hash =
1620            header::hash_from_scale_encoded_header(&fragment_to_verify.scale_encoded_header);
1621        let fragment_decoded_header = match header::decode(
1622            &fragment_to_verify.scale_encoded_header,
1623            self.inner.block_number_bytes,
1624        ) {
1625            Ok(j) => j,
1626            Err(err) => {
1627                self.inner.verify_queue.clear();
1628                self.inner.warp_sync_fragments_download = None;
1629                return (self.inner, Err(VerifyFragmentError::InvalidHeader(err)));
1630            }
1631        };
1632        let fragment_decoded_justification = match decode::decode_grandpa_justification(
1633            &fragment_to_verify.scale_encoded_justification,
1634            self.inner.block_number_bytes,
1635        ) {
1636            Ok(j) => j,
1637            Err(err) => {
1638                self.inner.verify_queue.clear();
1639                self.inner.warp_sync_fragments_download = None;
1640                return (
1641                    self.inner,
1642                    Err(VerifyFragmentError::InvalidJustification(err)),
1643                );
1644            }
1645        };
1646
1647        // Make sure that the header would actually advance the warp sync process forward.
1648        if fragment_decoded_header.number <= self.inner.warped_header_number {
1649            self.inner.verify_queue.clear();
1650            self.inner.warp_sync_fragments_download = None;
1651            return (
1652                self.inner,
1653                Err(VerifyFragmentError::BlockNumberNotIncrementing),
1654            );
1655        }
1656
1657        // Make sure that the justification indeed corresponds to the header.
1658        if *fragment_decoded_justification.target_hash != fragment_header_hash
1659            || fragment_decoded_justification.target_number != fragment_decoded_header.number
1660        {
1661            let error = VerifyFragmentError::TargetHashMismatch {
1662                justification_target_hash: *fragment_decoded_justification.target_hash,
1663                justification_target_height: fragment_decoded_justification.target_number,
1664                header_hash: fragment_header_hash,
1665                header_height: fragment_decoded_header.number,
1666            };
1667            self.inner.verify_queue.clear();
1668            self.inner.warp_sync_fragments_download = None;
1669            return (self.inner, Err(error));
1670        }
1671
1672        // Check whether the justification is valid.
1673        if let Err(err) = verify::verify_justification(verify::JustificationVerifyConfig {
1674            justification: &fragment_to_verify.scale_encoded_justification,
1675            block_number_bytes: self.inner.block_number_bytes,
1676            authorities_list: finalized_triggered_authorities
1677                .iter()
1678                .map(|a| &a.public_key[..]),
1679            authorities_set_id: *after_finalized_block_authorities_set_id,
1680            randomness_seed,
1681        }) {
1682            self.inner.verify_queue.clear();
1683            self.inner.warp_sync_fragments_download = None;
1684            return (
1685                self.inner,
1686                Err(VerifyFragmentError::JustificationVerify(err)),
1687            );
1688        }
1689
1690        // Try to grab the new list of authorities from the header.
1691        let new_authorities_list = fragment_decoded_header
1692            .digest
1693            .logs()
1694            .find_map(|log_item| match log_item {
1695                header::DigestItemRef::GrandpaConsensus(grandpa_log_item) => match grandpa_log_item
1696                {
1697                    header::GrandpaConsensusLogRef::ScheduledChange(change)
1698                    | header::GrandpaConsensusLogRef::ForcedChange { change, .. } => {
1699                        Some(change.next_authorities)
1700                    }
1701                    _ => None,
1702                },
1703                _ => None,
1704            })
1705            .map(|next_authorities| {
1706                next_authorities
1707                    .map(header::GrandpaAuthority::from)
1708                    .collect()
1709            });
1710
1711        // Fragments must only include headers containing an update to the list of authorities,
1712        // unless it's the very head of the chain.
1713        if new_authorities_list.is_none()
1714            && (!fragments_to_verify.final_set_of_fragments
1715                || fragments_to_verify.next_fragment_to_verify_index
1716                    != fragments_to_verify.fragments.len() - 1)
1717        {
1718            self.inner.verify_queue.clear();
1719            self.inner.warp_sync_fragments_download = None;
1720            return (self.inner, Err(VerifyFragmentError::NonMinimalProof));
1721        }
1722
1723        // Verification of the fragment has succeeded 🎉. We can now update `self`.
1724        fragments_to_verify.next_fragment_to_verify_index += 1;
1725        self.inner.warped_header_number = fragment_decoded_header.number;
1726        self.inner.warped_header_state_root = *fragment_decoded_header.state_root;
1727        self.inner.warped_header_extrinsics_root = *fragment_decoded_header.extrinsics_root;
1728        self.inner.warped_header_hash = fragment_header_hash;
1729        self.inner.warped_header = fragment_to_verify.scale_encoded_header.clone(); // TODO: figure out how to remove this clone()
1730        self.inner.warped_block_ty = WarpedBlockTy::Normal;
1731        self.inner.runtime_download = RuntimeDownload::NotStarted {
1732            hint_doesnt_match: false,
1733        };
1734        if !matches!(self.inner.body_download, BodyDownload::NotNeeded) {
1735            self.inner.body_download = BodyDownload::NotStarted;
1736        }
1737        self.inner.runtime_calls =
1738            runtime_calls_default_value(self.inner.verified_chain_information.as_ref().consensus);
1739        if let Some(new_authorities_list) = new_authorities_list {
1740            *finalized_triggered_authorities = new_authorities_list;
1741            *after_finalized_block_authorities_set_id += 1;
1742        }
1743        if fragments_to_verify.next_fragment_to_verify_index == fragments_to_verify.fragments.len()
1744        {
1745            self.inner.verify_queue.pop_front().unwrap();
1746        }
1747
1748        // Returning.
1749        let result = Ok((
1750            self.inner.warped_header_hash,
1751            self.inner.warped_header_number,
1752        ));
1753        (self.inner, result)
1754    }
1755}

1757/// Error potentially returned by [`VerifyWarpSyncFragment::verify`].
1758#[derive(Debug)]
1759pub enum VerifyFragmentError {
1760    /// Justification found within the fragment is invalid.
1761    JustificationVerify(verify::JustificationVerifyError),
1762    /// Mismatch between the block targeted by the justification and the header.
1763    TargetHashMismatch {
1764        /// Hash of the block the justification targets.
1765        justification_target_hash: [u8; 32],
1766        /// Height of the block the justification targets.
1767        justification_target_height: u64,
1768        /// Hash of the header.
1769        header_hash: [u8; 32],
1770        /// Height of the header.
1771        header_height: u64,
1772    },
1773    /// Warp sync fragment doesn't contain an authorities list change when it should.
1774    NonMinimalProof,
1775    /// Header does not actually advance the warp syncing process. This means that a source has
1776    /// sent a header below the requested hash.
1777    BlockNumberNotIncrementing,
1778    /// Warp sync proof is empty.
1779    EmptyProof,
1780    /// Failed to decode header.
1781    InvalidHeader(header::Error),
1782    /// Failed to decode justification.
1783    InvalidJustification(decode::JustificationDecodeError),
1784}

1791                justification_target_hash,
1792                justification_target_height,
1793                header_hash,
1794                header_height,
1795            } => {

1797                    f,
1798                    "Justification target (hash: {}, height: {}) doesn't match the associated header (hash: {}, height: {})",
1799                    HashDisplay(justification_target_hash),
1800                    justification_target_height,
1801                    HashDisplay(header_hash),
1802                    header_height,
1803                )
1804            }
1805            VerifyFragmentError::NonMinimalProof => write!(
1806                f,
1807                "Warp sync proof fragment doesn't contain an authorities list change"

1906            };

1977                &hint.storage_value

1984        } else {
1985            match decoded_downloaded_runtime
1986                .storage_value(&self.inner.warped_header_state_root, b":code")

1988                Ok(Some((code, _))) => code,
1989                Ok(None) => {
1990                    self.inner.warped_block_ty = WarpedBlockTy::KnownBad;
1991                    self.inner.runtime_download = RuntimeDownload::NotStarted {
1992                        hint_doesnt_match: *hint_doesnt_match,
1993                    };
1994                    if !matches!(self.inner.body_download, BodyDownload::NotNeeded) {

1996                    }

1998                }
1999                Err(proof_decode::IncompleteProofError { .. }) => {

2003                        Err(BuildRuntimeError::SourceMisbehavior(SourceMisbehavior {

2006                        })),
2007                    );
2008                }
2009            }
2010        };
2011
2012        let finalized_storage_heappages = match decoded_downloaded_runtime
2013            .storage_value(&self.inner.warped_header_state_root, b":heappages")
2014        {
2015            Ok(val) => val.map(|(v, _)| v),
2016            Err(proof_decode::IncompleteProofError { .. }) => {
2017                let downloaded_source = *downloaded_source;
2018                return (
2019                    self.inner,
2020                    Err(BuildRuntimeError::SourceMisbehavior(SourceMisbehavior {
2021                        source_id: downloaded_source,
2022                        error: SourceMisbehaviorTy::MerkleProofEntriesMissing,
2023                    })),
2024                );
2025            }
2026        };
2027

2030                Ok(hp) => hp,
2031                Err(err) => {

2034                        hint_doesnt_match: *hint_doesnt_match,
2035                    };
2036                    if !matches!(self.inner.body_download, BodyDownload::NotNeeded) {
2037                        self.inner.body_download = BodyDownload::NotStarted;

2040                }
2041            };

2044            module: &finalized_storage_code,
2045            heap_pages: decoded_heap_pages,
2046            exec_hint,
2047            allow_unresolved_imports,
2048        }) {
2049            Ok(runtime) => runtime,
2050            Err(err) => {
2051                self.inner.warped_block_ty = WarpedBlockTy::KnownBad;
2052                self.inner.runtime_download = RuntimeDownload::NotStarted {
2053                    hint_doesnt_match: *hint_doesnt_match,
2054                };

2063            chain_information::build::Config {
2064                finalized_block_header: chain_information::build::ConfigFinalizedBlockHeader::Any {
2065                    scale_encoded_header: self.inner.warped_header.clone(),

2067                        None
2068                    } else {
2069                        Some(self.inner.warped_finality.clone())
2070                    },
2071                },
2072                block_number_bytes: self.inner.block_number_bytes,
2073                runtime,
2074            },
2075        );

2077        if let chain_information::build::ChainInformationBuild::InProgress(in_progress) =
2078            &chain_info_builder
2079        {

2086            }
2087        }
2088
2089        self.inner.runtime_download = RuntimeDownload::Verified {
2090            downloaded_runtime: DownloadedRuntime {
2091                storage_code: Some(finalized_storage_code.to_vec()),
2092                storage_heap_pages: finalized_storage_heappages.map(|v| v.to_vec()),
2093                code_merkle_value: Some(finalized_storage_code_merkle_value),
2094                closest_ancestor_excluding: Some(finalized_storage_code_closest_ancestor_excluding),
2095            },
2096            chain_info_builder,
2097        };
2098
2099        (self.inner, Ok(()))
2100    }
