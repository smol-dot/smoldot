Module sources

Source
Expand description

Collection of sources used for the all_forks syncing.

Each source stored in the AllForksSources is associated to:

  • A SourceId.
  • A best block.
  • A list of non-finalized blocks known by this source.
  • An opaque user data, of type TSrc.
