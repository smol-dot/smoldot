pub fn main_trie_diff_get(&self, key: &[u8]) -> Option<Option<&[u8]>>

Returns the change, if any, at the given key of the main trie.

Returns None if the runtime hasn’t modified this entry in the main trie, and Some(None) if the runtime has removed the storage item of this entry.

Source

pub fn main_trie_storage_changes_iter_unordered( &self, ) -> impl Iterator<Item = (&[u8], Option<&[u8]>)> + Clone

Returns an iterator to all the entries of the main trie that are modified by this runtime call.

Each value is either Some if the runtime overwrites this value, or None if it erases the underlying value.

Note: This function is equivalent to StorageChanges::storage_changes_iter_unordered, except that it only returns changes made to the main trie.

Source

pub fn tries_with_storage_changes_unordered( &self, ) -> impl Iterator<Item = &[u8]>

Returns an iterator to all the entries of the given child trie that are modified by this runtime call.

Each value is either Some if the runtime overwrites this value, or None if it erases the underlying value.

Note: This function is equivalent to StorageChanges::storage_changes_iter_unordered, except that it only returns changes made to the given child trie.
